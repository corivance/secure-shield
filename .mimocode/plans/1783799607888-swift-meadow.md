# UI Redesign Plan — Soft-Flat Minimalist SaaS Aesthetic

## Scope
Visual-only redesign of the entire frontend. Zero changes to functionality, business logic, routing, API calls, state management, or user flows.

## Current State Analysis

The app uses a `tailwind.config.js` with custom colors (ink, charcoal, gray, paleblue, beige, taupe, softgreen), custom fonts (Poppins display, Inter sans, JetBrains Mono), custom shadows (soft, lift), and custom CSS component classes in `index.css` (ss-card, ss-panel, ss-btn, ss-input, etc.). The current design language is "geometric minimalist" with glassmorphism (backdrop-blur, translucent backgrounds), grain texture overlays, gradient hairline rules, and taupe-heavy accents.

## Design Target
- Background: `#F8FAFC` (slate-50)
- Surface/cards: Pure white `bg-white`
- Primary accent: `#4F46E5` (indigo-600) — replaces taupe for interactive elements
- Shadows: Ultra-subtle `shadow-[0_4px_20px_rgba(0,0,0,0.02)]`
- Corners: `rounded-2xl`
- Typography: Clean hierarchy — bold titles, medium sections, muted secondary
- Spacing: Increased padding/margins throughout
- Interactive: Rounded, subtle hover, soft focus, smooth transitions

## Implementation Plan

### Phase 1: Tailwind Config + Global CSS (foundation)

**Files:**
- `frontend/tailwind.config.js`
- `frontend/src/index.css`

**Changes:**
1. Add `indigo` color to Tailwind config: `indigo: { DEFAULT: '#4F46E5', 50: '#EEF2FF', 100: '#E0E7FF', 500: '#6366F1', 600: '#4F46E5', 700: '#4338CA' }`
2. Update shadow tokens: replace `soft` and `lift` with the new subtle shadow `0 4px 20px rgba(0,0,0,0.02)`
3. In `index.css`:
   - Remove the body gradient background images and grain texture overlay
   - Set `body` background to `#F8FAFC`
   - Remove `::selection` styling (or update to indigo-tinted)
   - Update `.ss-card`: remove `backdrop-blur-xl`, remove `border border-gray/70`, use `bg-white rounded-2xl shadow-[0_4px_20px_rgba(0,0,0,0.02)]`
   - Update `.ss-panel`: remove backdrop-blur, use `bg-white rounded-2xl border border-gray-200`
   - Update `.ss-btn-primary`: change from `bg-taupe` to `bg-indigo-600`, hover to `bg-indigo-700`
   - Update `.ss-btn-secondary`: remove `bg-white/60`, use `bg-white`
   - Update `.ss-btn-ghost`: change hover to `hover:bg-indigo-50 hover:text-indigo-600`
   - Update `.ss-input`: remove `bg-white/70`, use `bg-white`, focus ring to indigo
   - Update `.ss-label`: keep as-is (mono uppercase)
   - Update `.ss-interactive`: change hover border to indigo
   - Update `.ss-dot`: keep softgreen
   - Update `.ss-rule`: simplify to a plain light gray line
   - Remove `.ss-tag` border/transparency styling, use solid white bg
